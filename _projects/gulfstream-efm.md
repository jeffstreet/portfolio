Gulfstream G650 Empennage Flutter Model is an actuated aeroelastic flutter wind tunnel model. Designed remote model positioning systems, synchronized model excitation systems, and associated control software. Supported these systems through validation testing and wind-on data collection in NASA Langley's Transonic Dynamics Tunnel.
